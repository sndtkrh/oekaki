# Oekaki

[Oekaki](https://sndtkrh.github.io/oekaki/build/) is a string diagram editor.

## Dependency
- Node.js >= v12.18
- yarn >= 1.22

## Build and Run

First, run
```
$ yarn install
```


Run the following commands on two terminals:
```
$ yarn re:watch
```

```
$ yarn start
```

and access to http://localhost:3000 .
